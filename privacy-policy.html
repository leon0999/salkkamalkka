살까말까(이하 "본 앱")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보처리방침은 본 앱이 수집하는 정보와 그 처리 방법에 대해 설명합니다.

1. 개인정보의 수집 및 이용 목적

본 앱은 어떠한 개인정보도 수집하지 않습니다.

살까말까는 회원가입이 필요 없으며, 이름, 이메일, 전화번호, 주소 등 어떠한 형태의 개인정보도 수집하지 않습니다.

2. 수집하는 개인정보 항목

항목 수집 여부 설명
이름, 이메일, 전화번호 ❌ 수집 안 함 회원가입 불필요
위치 정보 ❌ 수집 안 함 위치 권한 요청 안 함
기기 정보 ❌ 수집 안 함 분석 도구 사용 안 함
사용자 입력 데이터 ✅ 로컬 저장 기기에만 저장, 서버 전송 안 함
사진 ✅ 로컬 저장 기기에만 저장, 서버 전송 안 함

3. 데이터 저장 방식

3.1 로컬 저장 (Local Storage)

본 앱에서 사용자가 입력한 모든 데이터는 사용자의 기기에만 저장됩니다.

  • 저장 위치: iOS UserDefaults (기기 내부)
  • 저장 데이터:
    • 위시리스트 아이템 정보 (물건 이름, 가격, 메모)
    • 등록 날짜 및 대기 기간
    • 절약 통계 데이터
    • 물건 사진 (사용자가 선택한 경우)
  • 서버 전송: 없음
  • 외부 공유: 없음

3.2 사진 라이브러리 접근

위시리스트 물건 이미지 등록을 위해 사진 라이브러리 접근 권한을 요청합니다.

  • 사진은 사용자 기기에만 저장됩니다
  • 외부 서버로 전송되지 않습니다
  • 사용자가 직접 선택한 사진만 접근합니다
  • 원본 사진은 변경되지 않습니다

4. 개인정보의 보유 및 이용 기간

모든 데이터는 사용자 기기에만 저장되며, 다음과 같은 경우 삭제됩니다:

  • 앱 삭제 시: 모든 데이터가 함께 삭제됩니다
  • 아이템 삭제 시: 해당 아이템의 데이터만 삭제됩니다
  • 앱 재설치 시: 기존 데이터는 복구되지 않습니다 (백업 없음)

5. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

  • 서버 통신 없음
  • 외부 광고 SDK 사용 안 함
  • 분석 도구 사용 안 함
  • 제3자 서비스 연동 없음

6. 개인정보의 파기

본 앱은 서버에 개인정보를 저장하지 않으므로 별도의 파기 절차가 없습니다. 사용자가 앱을 삭제하면 모든 데이터가 자동으로 삭제됩니다.

7. 이용자 및 법정대리인의 권리

본 앱은 개인정보를 수집하지 않으므로, 다음과 같은 권리 행사가 해당되지 않습니다:

  • 개인정보 열람 요구권
  • 개인정보 정정·삭제 요구권
  • 개인정보 처리정지 요구권

모든 데이터는 사용자 기기에만 저장되므로, 사용자가 직접 앱 내에서 수정 및 삭제할 수 있습니다.
